
## 1. Get Started

#### Dependencies and Installation
* Python 3.13.3

1. Clone Repo
```
git clone https://github.com/RongRongJhang/HW3.git
```
2. Install Dependencies
```
cd HW3
pip install -r requirements.txt
```
3. Run Code
```
python3 main.py
```
